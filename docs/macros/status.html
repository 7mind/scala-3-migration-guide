Migration Status

While being experimental, the Scala community has largely adopted the Scala 2 Def Macro feature in multiple of ways: code generation, optimizations, ergonomic DSLs...

Here is an incomplete list of libraries that use Scala 2 macros and their migration status:

  • sourcecode – cross-compiles to Scala 2 and Scala 3
  • utest – cross-compiles to Scala 2 and Scala 3
  • intent - compiles to Scala 3
  • xml-interpolator - compiles to Scala 3
  • expression-evaluator - not migrated, no replacement for Evals.eval
  • jsoniter-scala - not migrated, no replacement for Evals.eval
  • minitest - ported to Scala 3 in the Dotty community build but not merged to upstream
  • munit - ported to Scala 3 in the Dotty community build but not merged to upstream
  • scalatest - ported to Scala 3 in the Dotty community build but not merged to upstream
  • scodec-bits - ported to Scala 3 in the Dotty community build but not merged to upstream
